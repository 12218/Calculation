# 四则运算生成器
## 工程文件说明
![01](https://img-blog.csdnimg.cn/20210502211425970.png#pic_center)

calculate.py       生成算式

graph.py       pyqt5的图形界面

resources文件夹        包含使用到的qss样式和图片

results         保存生成的算式及答案

## 效果展示
